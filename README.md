# Data Analyst & Machine Learning Portfolio

This repository showcases hands-on projects from the Machine Learning course, covering core machine learning concepts and practical implementations in Python. Each section contains Jupyter notebooks demonstrating algorithms, visualizations, and results.

---


### Part 1: Data Preprocessing
-  **Data Cleaning** ‚Äì Removing inconsistencies and handling missing values
-  **Encoding Categorical Data** ‚Äì Converting text labels into numerical format
-  **Splitting Train/Test Sets** ‚Äì Separating data for training and evaluation
-  **Feature Scaling** ‚Äì Standardizing feature ranges for optimal model performance

### Part 2: Regression
-  **Simple Linear Regression** ‚Äì Predicting salaries based on years of experience
-  **Multiple Linear Regression** ‚Äì Predicting startup profit based on marketing, admin, and R&D spending
-  **Polynomial Regression** ‚Äì Modeling non-linear relationships between position level and salary
-  **Support Vector Regression (SVR)** ‚Äì Predicting salaries with margin-based support vector technique
-  **Decision Tree Regression** ‚Äì Predicting salaries using decision boundaries
-  **Random Forest Regression** ‚Äì Predicting salaries with ensemble of decision trees for better accuracy

### Part 3: Classification
-  **Logistic Regression** ‚Äì Predicting whether a customer will purchase a product
-  **K-Nearest Neighbors (K-NN)** ‚Äì Classifying users based on social network ad data
-  **Support Vector Machine (SVM)** ‚Äì Classifying data using a maximum margin hyperplane
-  **Kernel SVM** ‚Äì Handling non-linear classification problems
-  **Naive Bayes** ‚Äì Predicting if users liked a social ad using probabilistic classifiers
-  **Decision Tree Classification** ‚Äì Visual decision boundaries to classify purchases
-  **Random Forest Classification** ‚Äì Improving classification accuracy with multiple trees

### Part 4: Clustering
-  **K-Means Clustering** ‚Äì Segmenting mall customers based on income and spending score
-  **Hierarchical Clustering** ‚Äì Building customer segments using a dendrogram approach

### Part 5: Association Rule Learning
-  **Apriori** ‚Äì Mining association rules for market basket analysis
-  **Eclat** ‚Äì Discovering itemsets based only on support metrics

### Part 6: Reinforcement Learning
-  **Upper Confidence Bound (UCB)** ‚Äì Choosing the best ad to display using exploration vs exploitation
-  **Thompson Sampling** ‚Äì Bayesian approach to ad optimization based on success/failure

### Part 7: Natural Language Processing
-  **Sentiment Analysis** ‚Äì Classifying restaurant reviews as positive or negative using NLP and Bag-of-Words

### Part 8: Deep Learning
-  **Artificial Neural Networks (ANN)** ‚Äì Predicting customer churn based on contract and activity data
-  **Convolutional Neural Networks (CNN)** ‚Äì Image classification of cats vs dogs

### Part 9: Dimensionality Reduction
-  **Principal Component Analysis (PCA)** ‚Äì Reducing features in wine dataset for classification
-  **Linear Discriminant Analysis (LDA)** ‚Äì Maximizing class separation before classification
-  **Kernel PCA** ‚Äì Applying non-linear dimensionality reduction techniques
